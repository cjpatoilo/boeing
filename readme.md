## Why it's awesome

Boeing provides an automated setup for a faster and secure deployment to GitHub Pages. Hope you enjoy!


## Install

**Install with npm**

```sh
$ npm i -g boeing
```

**Install with Yarn**

```sh
$ yarn global add boeing
```

**Run with npx (without installing)**

```sh
$ npx boeing
```


## Usage

```
$ boeing --help

  Usage:

    boeing <directory> [<options>]

  Options:

    -h, --help              Display help information about Boeing
    -v, --version           Output Bower version

  Examples:

    boeing
    boeing dist

```

Note: If there is no `directory` defined, dist is default
